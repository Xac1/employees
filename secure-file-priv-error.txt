Для Windows:

В провіднику зайти за шляхом:
C:\ProgramData\MySQL\MySQL Server 8.0
Папка ProgramData прихована, тому треба скопіювати звідси шлях
й вставити в провіднику

Знайти файл my.ini й відкрити його в блокноті.
В файлі знайти [mysqld]
й нижче добавити ось таку строку:
secure-file-priv = ""
Щоб стало ось так:
[mysqld]
secure-file-priv = ""
Перезапустити сервер (перезавантажити компьютер)

Для MacOS:

В Finder зайти за шляхом:
/usr/local/opt/mysql/
Там відкрити на редагування файл my.cnf
В файлі знайти [mysqld]
й нижче добавити ось таку строку:
secure-file-priv = ""
Щоб стало ось так:
[mysqld]
secure-file-priv = ""
Перезапустити сервер (перезавантажити компьютер)
